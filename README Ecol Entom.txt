DATA & FILE OVERVIEW

The data is stored in one .txt file

no-choice experiment and behav.txt

	soil: three types (categorical data)
	id: beetle id (categorical data)
	colony: ant colony id (categorical data)
	semiochemical: arena coated with semiochemicals or not (binary data)
	N: number of ignoring interaction (count data)
	I: number of inspecting interaction (count data)
	O: number of opening mandible interaction (count data)
	B: number of biting interaction (count data)
	catch: number of beetle catch (count data)
	bending: number of abdomen bending of beetle (count data)
	Agr: total number of aggressive interactions (count data)
	Nagr: total number of non-aggressive interactions (count data)
	Total: total number of  interactions (count data)
